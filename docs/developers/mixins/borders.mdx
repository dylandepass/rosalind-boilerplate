# Border Mixins

Rosalind provides the following mixin to work with borders.

## Border Default

The `border-default` mixin creates a default border for the brand. This includes a `sm` border width, a `md` border radius and the `base` border color.

```css
@mixin border-default;
```

## Border Color

The `border-color` mixin can be used to set a borders color.

```css
@mixin border-color {type};
```
