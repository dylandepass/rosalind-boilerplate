# Spacing Mixins

Rosalind provides the following mixins to work with paddings, margin and gutters.

## Padding Mixins

Each mixins includes a required `size` parameter that is used to determine this size of the padding to use.

### Mixins

Set the padding on all sides (padding-top, padding-right, padding-bottom, padding-left).

```css
@mixin padding {size};
```

Set the padding of an individual side.

```css
@mixin padding-top {size};

@mixin padding-bottom {size};

@mixin padding-left {size};

@mixin padding-right {size};
```

Set the vertical or horizontal padding.

```css
@mixin padding-vertical {size};

@mixin padding-horizontal {size};
```

#### Padding by breakpoint

Setting the padding based on a breakpoint is also supported.

```css
@mixin padding-{breakpoint} {size};

@mixin padding-{breakpoint}-top {size};

@mixin padding-{breakpoint}-bottom {size};

@mixin padding-{breakpoint}-left {size};

@mixin padding-{breakpoint}-right {size};

@mixin padding-{breakpoint}-vertical {size};

@mixin padding-{breakpoint}-horizontal {size};
```
