# gap

The `gap`, `gap-x` and `gap-y` mixins can be used to add spacing between grid/flex columns and rows.

## Gap

The `gap` mixin can be used to add uniform spacing between grid columns and rows.

<Story of={GridStories.Gap}/>

```html
<div class="grid-demo-gap">
    <div>col</div>
    <div>col</div>
    <div>col</div>
    <div>col</div>
    <div>col</div>
    <div>col</div>
    <div>col</div>
    <div>col</div>
</div>
```

```css
.grid-demo-gap {
    display: grid;
    @mixin grid-cols 4;
    @mixin gap 2xl;

    & > div {
        @mixin layer-1;
        text-align: center;
    }
}
```
### Responsive Gap

The `gap` can be adjusted by breakpoint using the responsive versions of the `gap` mixins.

```css
@mixin gap-{breakpoint} {size};
```
