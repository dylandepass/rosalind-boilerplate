# grid-col-start

The `grid-col-start` mixin can be used to specifies a grid items start position within the grid.

```css
@mixin grid-col-start {position};
```

### Responsive Grid Column Start

The `grid-col-start` mixin can be adjusted by breakpoint using the responsive version.

```css
@mixin grid-{breakpoint}-col-start {position};
```
