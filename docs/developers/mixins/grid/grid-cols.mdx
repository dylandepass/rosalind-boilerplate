# grid-cols

The `grid-cols` mixin can be used to create a CSS grid with the provided number of columns. 

```css
@mixin grid-cols {columns};
```

### Responsive Grids

The `grid-cols` can be adjusted by breakpoint using the responsive versions of the `grid-cols` mixins.

```css
@mixin grid-{breakpoint}-cols {columns};
```

### Example with 8 responsive columns

The demo below shows how to implement as responsive grid that is 8 columns on large screens, 4 columns on medium screens, 2 columns on small screens and 1 column for everything else.

<Story of={GridStories.ResponsiveColumns}/>

```html
<div class="grid-demo-responsive">
    <div>col</div>
    <div>col</div>
    <div>col</div>
    <div>col</div>
    <div>col</div>
    <div>col</div>
    <div>col</div>
    <div>col</div>
</div>
```

```css
.grid-demo-responsive {
    display: grid;
    @mixin grid-cols 1;
    @mixin grid-sm-cols 2;
    @mixin grid-md-cols 4;
    @mixin grid-lg-cols 8;
    @mixin gap xs;
    & > div {
        @mixin layer-1;
        text-align: center;
    }
}
```
